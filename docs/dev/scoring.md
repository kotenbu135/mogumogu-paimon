# スコア計算方式

実装: `webapp/src/lib/scoring.ts`

## スコアタイプ一覧

| スコアタイプ | 計算式 |
|---|---|
| CV | 会心率×2 + 会心ダメージ |
| HP型 | CV + HP%×1.0 |
| 攻撃型 | CV + 攻撃力%×1.0 |
| 防御型 | CV + 防御力%×0.8 |
| 熟知型 | CV + 元素熟知×0.25 |
| チャージ型 | CV + 元素チャージ×0.9 |
| 最良型 | 上記6タイプの最高値 |

定義: `webapp/src/lib/scoring.ts:45-51`

## 再構築成功率

実装: `webapp/src/lib/reconstruction.ts`

再構築によりスコアが向上する確率を多項分布ベースで算出する。

### 計算手順

1. 強化N回を4サブステに振り分ける全パターンを列挙（多項分布）
2. 保証条件（選択2サブステへの合計ロール数 >= 閾値）でフィルタし確率を正規化
3. 各パターンの再構築後スコアを期待値ベース（`AVG_INCREMENT × ロール数`）で算出
4. 現在スコアを上回るパターンの正規化確率を合計 → 成功率（%）

### 再構築種別と保証閾値

| 種別 | 保証閾値 |
|------|----------|
| 通常再構築 | 合計 2回以上 |
| 上級再構築 | 合計 3回以上 |
| 絶対再構築 | 合計 4回以上 |

### 保証サブステの自動選定

| スコアタイプ | 保証サブステ |
|---|---|
| CV / 最良型 | 会心率 + 会心ダメージ |
| HP型 | 会心率 + HP%（または会心ダメージ + HP%） |
| 攻撃型 | 会心率 + 攻撃力%（または会心ダメージ + 攻撃力%） |
| 防御型 | 会心率 + 防御力%（または会心ダメージ + 防御力%） |
| 熟知型 | 会心率 + 元素熟知（または会心ダメージ + 元素熟知） |
| チャージ型 | 会心率 + 元素チャージ効率（または会心ダメージ + 元素チャージ効率） |

複数候補がある場合は最も成功率が高いペアを自動選択する。

### 主要エクスポート関数

| 関数 | 説明 |
|---|---|
| `calculateReconstructionRate` | 再構築成功率を算出（0〜100）。対象外は null |
| `enumeratePatterns` | N回をK箱に分配する全パターン列挙 |
| `multinomialProb` | 多項分布の確率計算 |
| `getGuaranteedIndices` | 保証サブステのインデックスペア取得 |
