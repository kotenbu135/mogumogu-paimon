# mogumogu-paimon

原神の聖遺物スコアを GOOD フォーマットから計算・ランキング表示する Web アプリ。

## スコア計算方式

ドロップ画面でも確認できる各スコア計算式は以下の通り。

| スコアタイプ | 計算式 |
|---|---|
| **CVスコア** | 会心率×2 + 会心ダメージ |
| **HP型** | CV + HP%×1.0 |
| **攻撃型** | CV + 攻撃力%×1.0 |
| **防御型** | CV + 防御力%×0.8 |
| **熟知型** | CV + 元素熟知×0.25 |
| **チャージ型** | CV + 元素チャージ×0.9 |

各型スコアは「CVスコア + 最有力サブステータス × 係数」の合計値。
スコアが高いほど、そのビルドへの貢献度が高い聖遺物。

## 使い方

1. [Inventory Kamera](https://github.com/Andrewthe13th/Inventory_Kamera) 等で GOOD フォーマット JSON をエクスポート
2. ドロップ画面に JSON ファイルをドラッグ＆ドロップ（またはクリックして選択）
3. スコアタイプ・セット・部位でフィルタしてランキングを確認
