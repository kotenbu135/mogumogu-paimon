# Next.js Quality Gate

webapp/ 配下の Next.js プロジェクトに対して、以下の品質チェックをすべて実行してください。
各ステップでエラーが発生した場合は修正し、すべてのチェックが通るまで繰り返してください。

## 1. TypeScript 型チェック

```bash
bash -i -c "cd /home/sakis/genshin-artifacts/webapp && npx tsc --noEmit"
```

- エラー 0 件であること
- 型の不整合、未使用インポート、any の濫用がないか確認

## 2. Next.js ビルド

```bash
bash -i -c "cd /home/sakis/genshin-artifacts/webapp && npm run build"
```

- ビルドが正常に完了すること
- ビルド警告がある場合は内容を報告し、対応可能なものは修正

## 3. コード品質レビュー

`webapp/src/` 配下の全 `.ts` `.tsx` ファイルを読み取り、以下を確認:

- **未使用のインポート・変数** がないこと
- **`any` 型** の使用を避け、適切な型定義がされていること
- **ハードコードされた秘密情報** (APIキー等) が含まれていないこと
- **`console.log`** がデバッグ目的で残っていないこと
- **React Hook のルール違反** (条件付き Hook 呼び出し等) がないこと
